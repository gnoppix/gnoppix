About Gnoppix



Early Years
Gnoppix linux live cd was developed in June 2003, and was based upon Debian GNU/Linux 3.0 (woody). When it was first released, Gnoppix could be compared to Knoppix, with the difference being that Gnoppix used GNOME as the desktop GUI environment. After a successful implementation of the Gnoppix live cd functionality which flowed into Ubuntu in 2004.

Kubuntu was born on December 2004 at the Ubuntu Mataro Conference in Mataró, Spain. Gnoppix Project lead Andreas Mueller, got the approval from Mark Shuttleworth to start the first Ubuntu variant, called Kubuntu. Because of some difficulties, disagreements regarding how the free Kubuntu Project was leaded, Andreas stopped his work on Ubuntu/Kubuntu and J.Riddell took over.

Today
With the relaunch of Gnoppix during Covid-19 in 2021, the decision was made to move to KDE for the GUI, due to its flexibility to run both as a desktop and as a mobile environment. None of the existing distributions covered after almost 20 years of development the useability of average desktop users, which is I think very poor work after all this time. I'm older today, hehe, but the world is older too, today we using my Desktops/Laptops in another way how we did before. I'm heavily using a Smartphone for some tasks. And almost many other do it the same way. But when i saw Kubuntu I thought its still the same as I left the project. Unfortunately, the current Linux Distributions doesn't provide a useable desktop out of the box. Another point was the thread from Gnome and Debian against FSF founder RMS. I do not want support software from this cancel culture. And here we are again.

About Us

Welcome to Gnoppix

Based on the most recent Debian release with additions from Kali Linux , Gnoppix comes complete with a set of additional software repositories containing the latest 64-bit versions of the Plasma 5 desktop environment/framework, Qt 5 toolkit and other compatible KDE software Gnoppix was first announced in June 2003

In Addition to the famouse Gnoppix Desktop, we'll provide also a docker version of Gnoppix, currently we're working on a Gnoppix version for Mobile devices like smartphones and tablets as well.
